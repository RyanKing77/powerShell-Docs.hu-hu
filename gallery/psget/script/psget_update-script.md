# <a name="update-script"></a>Update-Script

Frissítés-parancsfájl parancsmag lehetővé teszi az helybeni frissítése a parancsfájlok, amelyek megtörtént-e telepítési parancsfájl-parancsmag használatával.

## <a name="description"></a>Leírás

A frissítés-parancsfájl parancsmag frissíti a megadott parancsfájl, amelyen korábban telepítették a tárházból.

## <a name="cmdlet-syntax"></a>A parancsmag szintaxisa

```powershell
Get-Command -Name Update-Script -Module PowerShellGet -Syntax
```
## <a name="cmdlet-online-help-reference"></a>A parancsmag online Súgó-hivatkozás

[Update-Script](http://go.microsoft.com/fwlink/?LinkId=619787)
