# <a name="the-windows-powershell-ise-scripting-object-model"></a>A Windows PowerShell ISE Scripting Hálózatiobjektum-modellje
  A Windows PowerShell® integrált parancsfájl-kezelési környezet (ISE) funkciókat, hogy az alapul szolgáló parancsfájl-kezelési hálózatiobjektum-modellje, amely lehetővé teszi a felhasználók számára a Windows PowerShell parancsfájlokat írhat, amelyek kezelhetők a különböző vizuális és funkcionális szempontjait keresztül elérhetővé teszi a A Windows PowerShell ISE. Parancsfájl-kezelési objektumok, a tulajdonságok és azok a módszerek vannak definiálva kifejezetten erre a célra.
