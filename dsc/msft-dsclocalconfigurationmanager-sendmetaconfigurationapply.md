# <a name="sendmetaconfigurationapply-method-of-the-msftdsclocalconfigurationmanager-class"></a>A MSFT_DSCLocalConfigurationManager osztály SendMetaConfigurationApply módszer

Beállítja a helyi Configuration Manager beállítások konfigurációs ügynök használt.

<a name="syntax"></a>Szintaxis
------

```mof
uint32 SendMetaConfigurationApply(
  [in] uint8   ConfigurationData[],
  [in] boolean force
);
```

<a name="parameters"></a>Paraméterek
----------

*ConfigurationData* \[a\]  
A környezet konfigurációjának adatait.

*kényszerített* \[a\]  
**Igaz** kényszerítése a konfigurációját, és állítsa le.

## <a name="return-value"></a>Visszatérési érték
------------

Sikeres művelet; nulla értéket ad vissza Ellenkező esetben hibakódot.

## <a name="remarks"></a>Megjegyzések

Ez a statikus módszer.

## <a name="requirements"></a>Követelmények
------------
>**MOF:** DscCore.mof

>**Namespace**: Root\Microsoft\Windows\DesiredStateConfiguration
