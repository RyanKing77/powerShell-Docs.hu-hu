# <a name="disabledebugconfiguration-method-of-the-msftdsclocalconfigurationmanager-class"></a>A MSFT_DSCLocalConfigurationManager osztály DisableDebugConfiguration módszer

Letiltja a DSC-erőforrás hibakeresés.

<a name="syntax"></a>Szintaxis
------

```mof
uint32 DisableDebugConfiguration();
```

<a name="parameters"></a>Paraméterek
----------

Ez a módszer nem paraméterrel rendelkezik.

## <a name="return-value"></a>Visszatérési érték
------------

Sikeres művelet; nulla értéket ad vissza Ellenkező esetben hibakódot.

## <a name="remarks"></a>Megjegyzések

Ez a statikus módszer.

## <a name="requirements"></a>Követelmények
------------
>**MOF:** DscCore.mof

>**Namespace**: Root\Microsoft\Windows\DesiredStateConfiguration
