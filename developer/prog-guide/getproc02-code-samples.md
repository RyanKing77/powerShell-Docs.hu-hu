# <a name="getproc02-code-samples"></a>GetProc02 – Kódminták

Az alábbiakban a GetProc02 minta parancsmag Kódminták. Ez a `Get-Process` parancsmag minta ismertetett [a folyamat parancssori bemenet-paramétereket adunk hozzá](../cmdlet/adding-parameters-that-process-command-line-input.md). Ez `Get-Process` parancsmag lekérdezi a nevük alapján folyamatok, majd megjeleníti a a folyamatokkal kapcsolatos információkat a parancssorból.

> [!NOTE]
> Letöltheti a C# forrásfájl (getproc02.cs) a Get-Proc parancsmag használatával a Microsoft Windows szoftverek fejlesztési Kit for Windows Vista és a .NET Framework 3.0 futtatási összetevői. Letöltési útmutatás: [Windows PowerShell telepítése és a Windows PowerShell SDK letöltési](/powershell/developer/installing-the-windows-powershell-sdk).
>
> A letöltött forrásfájlok érhetők el a  **\<PowerShell-minták >** könyvtár.
